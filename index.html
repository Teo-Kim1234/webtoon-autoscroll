필수 시스템 권한

다른 앱 위에 그리기 & 접근성 서비스 권한

1. PC & 모바일 브라우저용 확장 프로그램

아래의 다운로드 버튼을 클릭하면 크롬 확장 프로그램 완제품 설치 파일이 즉시 내 컴퓨터에 저장됩니다!

크롬 확장 프로그램 설치 패키지 다운로드 (ZIP)
설치 및 실행 1초 매뉴얼
  1. 다운로드한 ZIP 파일의 압축을 풀어줍니다.
  2. 크롬 브라우저에서 주소창에 chrome://extensions 입력 후 이동
  3. 우측 상단 [개발자 모드] 스위치를 켭니다.
  4. 좌측 상단 [압축해제된 확장 프로그램을 로드] 버튼을 클릭합니다.
  5. 압축을 푼 폴더를 선택하면 즉시 설치 끝!

2. 안드로이드 모바일 공식 앱용 자동 스크롤러

네이버웹툰 앱 위에서 오버레이로 바로 실행 가능한 Kotlin 네이티브 앱 소스입니다.

안드로이드 앱 개발 소스 패키지 다운로드 (ZIP)
폰에 집어넣는 빌드 매뉴얼
  1. 다운로드한 ZIP 파일의 압축을 풀어줍니다.
  2. Android Studio를 실행하고 [Open]을 누릅니다.
  3. 압축을 푼 폴더를 엽니다.
  4. 스마트폰을 PC에 연결하고, 상단의 **실행(▶)**을 누르면 APK 빌드 후 폰에 자동 설치됩니다!
코드가 클립보드에 복사되었습니다!
