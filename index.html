필수 시스템 권한

다른 앱 위에 그리기 & 접근성 서비스 권한

1. PC & 모바일 브라우저용 확장 프로그램

아래의 다운로드 버튼을 클릭하면 크롬 확장 프로그램 완제품 설치 파일이 즉시 내 컴퓨터에 저장됩니다!

크롬 확장 프로그램 설치 패키지 다운로드 (ZIP)
설치 및 실행 1초 매뉴얼
  1. 다운로드한 ZIP 파일의 압축을 풀어줍니다.
  2. 크롬 브라우저에서 주소창에 chrome://extensions 입력 후 이동
  3. 우측 상단 [개발자 모드] 스위치를 켭니다.
  4. 좌측 상단 [압축해제된 확장 프로그램을 로드] 버튼을 클릭합니다.
  5. 압축을 푼 폴더를 선택하면 즉시 설치 끝!

2. 안드로이드 모바일 공식 앱용 자동 스크롤러

스마트폰의 공식 네이버웹툰 앱 위에서 백그라운드 위젯 오버레이로 작동하는 프리미엄 자동 스크롤 앱입니다.

즉시 설치용 APK 완제품 다운로드 (추천!) 개발 소스 패키지 다운로드 (ZIP)
📱 초정밀 안드로이드 네이티브 앱 빌드 & 설치 가이드
1단계: ZIP 파일 다운로드 및 준비 압축 패키지(ZIP)를 다운로드한 후, 경로 내 한글이나 띄어쓰기가 없는 영문 폴더 경로(예: C:\projects\webtoon-scroller-android)에 압축을 풀어줍니다.
2단계: 스마트폰 개발자 모드 및 USB 디버깅 활성화 스마트폰 기기 자체에서 앱 개발자 권한을 승인해 주어야 설치할 수 있습니다:
  • 스마트폰 [설정] → [휴대전화 정보] → [소프트웨어 정보]로 들어갑니다.
  • [빌드 번호 (Build Number)]를 연속으로 **7번 연타**합니다. (화면에 '개발자 모드가 활성화되었습니다'라는 문구가 뜹니다.)
  • [설정] 메인 화면으로 돌아가 맨 아래로 스크롤하면 생긴 [개발자 옵션]에 들어갑니다.
  • 스크롤을 살짝 내려 [USB 디버깅] 스위치를 활성화해 줍니다.
3단계: 스마트폰 PC 연결 및 권한 수락 USB 케이블을 통해 스마트폰과 컴퓨터를 연결합니다. 스마트폰 화면에 **'USB 디버깅을 허용하시겠습니까?'**라는 보안 질문 팝업이 뜨면 [허용] 또는 [항상 허용]을 선택합니다.
4단계: Android Studio 프로젝트 임포트 및 빌드
  • Android Studio를 실행하고 [Open]을 클릭합니다.
  • 압축을 푼 폴더 내부의 webtoon-scroller-android 루트 폴더를 골라 열어줍니다.
  • 하단의 Gradle Build 연산이 모두 끝날 때까지 1~2분 대기합니다.
  • 프로젝트 창 상단에 내 스마트폰 기기명(예: Samsung SM-G998N 등)이 활성화되어 선택되었는지 확인합니다.
  • 우측 상단의 초록색 재생 모양 아이콘 [Run 'app' (▶)] 버튼을 클릭하면 컴파일 후 스마트폰에 앱이 자동 설치 및 실행됩니다!
최초 실행 시 시스템 권한 승인 요령 본 앱은 화면 위에 오버레이 위젯을 띄우고 제스처를 시뮬레이트하므로 반드시 최초 구동 시 다음 2가지 시스템 특별 승인이 필요합니다:
  1. 다른 앱 위에 그리기 권한 (Overlay Window): 앱 실행 시 나타나는 권한 팝업을 눌러 권한을 허용합니다.
  2. 접근성 권한 (Accessibility Service): 폰 [설정] → [접근성] → [설치된 앱] 또는 [설치된 서비스] → [웹툰 자동 스크롤러]를 찾아 '사용 안 함'을 '사용 중'으로 변경하여 제스처 주입 권한을 위임합니다.
